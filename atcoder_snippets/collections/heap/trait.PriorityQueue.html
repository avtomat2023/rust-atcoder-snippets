Priority queue trait.

Providing this as a trait is useful for concise coding. When you write a function receiving a MaxHeap, you don't have to write a trait bound for comparison function. If you write type directly, the signature looks like:

use std::cmp::Ordering;

fn f(heap: &mut MaxHeap<i32, impl Fn(&i32, &i32) -> Ordering>) {
    /* ... */
}

Thanks to PriorityQueue trait, you can write as:

fn f(heap: &mut impl PriorityQueue<i32>) {
    /* ... */
}
