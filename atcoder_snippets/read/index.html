[−][src]Module atcoder_snippets::read

Macros and functions for reading problem input.

Rustで競技プログラミングを戦うための最初の関門が、入力の読み込みである。 標準入力から数値を読み込むには、以下のコードを書く必要がある。

use std::io;

let mut line = String::new();
io::stdin().read_line(&mut line).unwrap();
// 末尾の改行文字を除去する
let line = line.trim_right();
let n = line.parse::<i32>();

このようなボイラープレートを問題ごとに記述するのは現実的でないため、 このモジュールの提供する関数・マクロを用いて、 入力読み込みを省力化するのが賢明である。

入力は行単位で読むよう設計されており、一回の関数呼び出しまたはマクロ展開で、 標準入力の1行または複数行を読み込む。

  • 標準入力の1行を読むには、readマクロを用いる。
  • 標準入力の複数行を読むには、read_chunkマクロを用いる。
  • 標準入力の一様な行を一定行数、あるいはすべて読むには、read_lines関数を用いる。
  • 標準入力の複数行に渡る入力を繰り返し読むには、read_chunks関数を用いる。 Typically useful for reading queries in Codeforces.
