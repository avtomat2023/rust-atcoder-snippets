A number whose arithmetics is carried modulo a prime number.

pub unsafe fn set_mod(modulus: ModPModulus) -> Result<(), String>[src]

Sets the modulus.

If modulus is not a prime number, returns Err.

Undefined behaviors

If you make another call of set_mod after creating ModP numbers, you must not use the numbers. The correctness of calculations using the numbers is not guaranteed.

If you call set_mod when two or more threads use ModP numbers, the correctness of calculations using the numbers is not guaranteed.

Calculate power using exponentiation by squaring.
